- TFLite는 학습된 모델을 모바일 환경에서 빠르게 추론할 수 있도록 경량화해주는 도구이며, 양자화된 모델은 용량과 속도 측면에서 유리하지만 성능이 약간 저하될 수 있다는 점을 이해했다.
- TFLite 모델과 함께 .tflite 파일과 labels.txt를 함께 배포해야 추론 결과를 해석할 수 있다는 점도 배웠다.
